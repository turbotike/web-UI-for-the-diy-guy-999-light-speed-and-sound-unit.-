Features

🖥️

Web-Based Configurator

Full browser GUI — adjust every setting with sliders, dropdowns, and checkboxes. No code editing.

🔊

Live Sound Builder

Load, preview, trim, crossfade, and smooth engine sounds in real-time. Install with one click.

🎚️

80+ Vehicle Profiles

Scania V8, Mack, Kenworth, Caterpillar, Defender, Mustang, LaFerrari, tanks, tractors, and more.

⚡

One-Click Setup

Double-click START.bat — Python installs automatically if needed, browser opens, done.

🎵

WAV → Header Converter

Drag & drop .wav files to create ESP32-compatible sound headers with resampling & normalization.

🚀

Compile & Flash

Build and upload firmware to your ESP32 directly from the web interface.

💾

Backup & Restore

Export your entire vehicle config. Share it, import it on another board, or roll back changes.

🔁

Seamless Loop Crossfade

Firmware-level 10% crossfade for perfectly seamless engine sound loops — no clicks or pops.

🎛️

Software Volume Override

Bypass the hardware volume pot entirely — control volume from the web slider (0-250%).
